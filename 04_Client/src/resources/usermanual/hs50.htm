Paranoid Agent

This is a "blind" agent that only knows its own stocking levels. It will function in both Push and Pull simulations.

 

Paranoid is the simplest of all the agents, whose only objective is to always remain fully stocked. This agent does not care about shipping costs nor does it care that its supplier may become backlogged.

 

This help document provides a qualitative comparison of the agents' performance.
