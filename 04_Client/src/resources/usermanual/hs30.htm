Starting the SCCORI Agents 

Start your browser and enter the following URL:

http://sccori-agent.francoisnadeau.com/

 

You should see the following page.

 

 

Click on the "Take me to the agent management window" link.

 

The SCCORI Agent Management page is divided into two sections.

 

The upper section

  • Is somewhat a reflection of the SCCORI simulation, but appears to lag it by one cycle.
  • It also shows the status of the agents; such as Agent Type, Role (e.g., retailer, wholesaler, distributor, or manufacturer), and Wait time.
  • Long wait times indicate that an agent is no longer being used. A RIP button to the right of each agent allows you to kill (and free up) agents that are no longer being used. (Be courteous and do not kill an other user's active agents.)

 

The lower section

  • Allows the you to enter the names and passwords of the unavailable players. These are the names and passwords that were used to create the SCCORI simulation meeting room. If you do not know the names and passwords of the other players, then just add your own agent player names to the SCCORI simulation.
  • Once the agent is logged in, the Select Meeting Room drop-down list becomes active and you can select the desired Meeting Room.
  • Select one of the available agents from the Select Agent drop down list and the Start Playing button becomes active. You may then click the Start Playing button.
  • Refer to the Agent Comparisons page to assist your selection of an appropriate agent.
