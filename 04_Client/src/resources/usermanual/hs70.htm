Rules Based (Push) Agent

This is a "blind" agent that only knows its own stocking levels. It will function in both Push and Pull simulations.

 

Rules Based (Push) agent does its best to maintain "adequate" stocking levels. This agent tries to minimize shipping costs and it avoids back logging its supplier - as long as the supplier is attempting to maintain adequate stocks itself.

 

The Rules Based (Push) Agent uses the following parameters:

  • maximum inventory depends on the position played
  • retailer = 90 (good delivery here minimizes backlogs overall)
  • wholesaler = 45 (tends to minimize propagation of backlogs)
  • distributor = 80
  • manufacturer = 80
  • restocking trigger level = 55 (order more product if SOH < 55)
  • container size = 60
  • minimum order = 30 (order half a container or more)

 

This agent does better than the Paranoid and Considerate Paranoid agents on Random runs, but it is the worst of all agents on Spiked runs.
