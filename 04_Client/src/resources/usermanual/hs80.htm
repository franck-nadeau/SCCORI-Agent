Rules Based (Generic) Agent

This is a "seeing" agent that knows all of the other players stocking levels. It should only be used in Pull simulations.

 

Rules Based (Generic) agent does its best to maintain "adequate" stocking levels. This agent tries to minimize shipping costs and it avoids back logging its supplier - as long as the supplier is attempting to maintain adequate stocks itself.

 

The Rules Based (Push) Agent uses the following parameters:

  • maximum inventory = 100 (it does not depend on the position played)
  • restocking trigger level = 50 (order product only if SOH < 50)
  • container size = 60
  • minimum order = 30 (order half a container or more)

 

This agent is the best overall option for Pull Random runs. It is the second-best agent for both Random and Spiked simulations. If you absolutely need the best, then consider the Rules Based (Pull Random) and Rules Based (Pull Spiked) agents respectively.
