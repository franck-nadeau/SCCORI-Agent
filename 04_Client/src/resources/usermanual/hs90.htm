Rules Based (Pull Random) Agent

This is a "seeing" agent that knows all of the other players stocking levels. It should only be used in Pull simulations.

 

Rules Based (Pull Random) agent uses the same rules as the Rules Base (Generic) agent, but has different parameter values. It does its best to maintain "adequate" stocking levels. This agent tries to minimize shipping costs and it avoids back logging its supplier - as long as the supplier is attempting to maintain adequate stocks itself.

 

The Rules Based (Push) Agent uses the following parameters:

  • maximum inventory depends on the position played
  • retailer = 100
  • wholesaler = 80
  • distributor = 80
  • manufacturer = 65 (higher previous stocking levels allow lower levels here)
  • restocking trigger level = 40 (order product if SOH < 40)
  • container size = 60
  • minimum order = 30 (order half a container or more)

 

This agent is the best option for Random Pull runs; but, it performs only marginally better than the Rules Based (Generic) agent. Its performance on Spiked runs is moderate, and only marginally better than Considerate Paranoid's performance there.
