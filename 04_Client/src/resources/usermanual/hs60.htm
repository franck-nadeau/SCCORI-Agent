Considerate Paranoid Agent

This is a "seeing" agent that knows all of the other players stocking levels. It should only be used in Pull simulations.

 

Considerate Paranoid is a simple agent, whose primary objective is to always remain fully stocked. This agent does not care about shipping costs; but, it will not cause its supplier to become backlogged. It will however, backlog its supplier if it determines that the supplier is not attempting to maintain its own stocking levels.

 

This help document provides a qualitative comparison of the agents' performance.
