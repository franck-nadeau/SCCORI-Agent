Overview

  • SCCORI is the acronym for Supply Chain Collaboration Online Research Institute.
  • The SCCORI site has a Supply Chain Simulation that normally requires four people to play.
  • SCCORI Agents allow people to run the simulation with less than four people by making use of Intelligent Agents to play some of the positions.

 

The SCCORI Agent web site provides one, two, or three intelligent agents to users of the SCCORI simulations. This is useful for people who wish to run simulations with less than four players.

 

A selection of agents is available, displaying simple to complex strategies. One agent performs exceptionally well in spiked pull simulations. Each of these agents utilize a consistent strategy across all simulations.

 

A single user could select three great (or awful) agents to see how various strategies in the remaining position perform against such agents. This help document provides a qualitative comparison of the agents' performance.
